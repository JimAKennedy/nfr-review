## How This Site Is Organised

**Learn** — A sequential training path. Start here. Each chapter builds on the last, from "what does this tool do?" through the full pipeline architecture to writing your own rules and collectors.

**Reference** — Jump-in, look-up material. CLI flags, config schema, the full rule catalogue, payload field definitions. Use this day-to-day.

**Recipes** — Task-oriented guides. "I need to scan a monorepo." "I need to set up baseline tracking." Step-by-step instructions for specific workflows.

**Appendices** — Architecture diagrams of nfr-review itself, a glossary of terms, and FAQ.
